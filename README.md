# Simple-Ftp-Server

pyftpdlib을 사용한 간단한 ftp 서버

filezilla 서버도 사용하기 싫어서 만든 스크립트

실행하고 On버튼만 누르면 현재 폴더(또는 지정폴더)가 ftp 서버가 됨.
